# Dulce Fé - Landing Page

Bienvenido al repositorio de la landing page de **Dulce Fé**.

## Descripción
Este proyecto es una landing page moderna y elegante para una marca de postres, diseñada para atraer a los amantes de lo dulce con una estética visual impactante.

## Tecnologías
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Estructura del Proyecto
- `index.html`: Estructura principal.
- `style.css`: Estilos y animaciones.
- `script.js`: Interactividad.
- `assets/`: Imágenes y recursos multimedia.

## Cómo ver el proyecto
Simplemente abre el archivo `index.html` en tu navegador web favorito.
